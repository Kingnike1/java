# Laboratório de Programação em Java ☕

Bem-vindo ao repositório de estudos e exercícios de Java. Este projeto contém uma coleção organizada de exercícios, exemplos e pequenos projetos desenvolvidos durante o aprendizado da linguagem Java, focando em lógica de programação, orientação a objetos e manipulação de dados.

## 📂 Estrutura do Repositório

O repositório está organizado de forma a facilitar a navegação entre os diferentes tópicos estudados:

| Diretório | Descrição |
| :--- | :--- |
| `src/exercicios/` | Exercícios práticos divididos por aulas (Aula 02 a 05). |
| `src/exemplos/` | Códigos de exemplo e demonstrações básicas (Hello World, etc). |
| `src/projetos/` | Pequenos projetos mais estruturados, como sistemas de cadastro. |
| `src/testes/` | Scripts de teste para validação de ambiente. |

## 🚀 Conteúdos Principais

### 1. Exercícios por Aula
- **Aula 02 & 03:** Fundamentos, variáveis e estruturas de controle iniciais.
- **Aula 04:** Estruturas de repetição (`while`, `for`), manipulação de Strings e coleções simples (`HashMap`).
- **Aula 05:** Trabalhando com matrizes e geração de números aleatórios.

### 2. Projeto de Cadastro
Localizado em `src/projetos/cadastro`, este é um exemplo prático de:
- Entrada de dados com `Scanner`.
- Validação de inputs (Regex).
- Organização de classes e objetos.

## 🛠️ Como Executar

Para compilar e executar qualquer arquivo Java neste repositório, você precisará do **JDK (Java Development Kit)** instalado em sua máquina.

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Kingnike1/java.git
   ```

2. **Navegue até o arquivo desejado:**
   ```bash
   cd src/exercicios/aula04
   ```

3. **Compile o arquivo:**
   ```bash
   javac A04t01.java
   ```

4. **Execute o programa:**
   ```bash
   java A04t01
   ```
